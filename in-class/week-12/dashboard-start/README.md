To check active port usin `ss` command in Linux
```
ss -nlp | grep :3000
```

To kill process ID, use 
```
kill ProcessIdNum
```

If the process refuses to exit, the just use the `-9` flag, which is   
a `SIGTERM` and cannot be ignored
```
kill -9 ProcessIdNum
```
